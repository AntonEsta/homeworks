1. Разработать программу – картотеку домашних животных.
    У каждого животного есть: уникальный идентификационный номер,
    кличка, хозяин (объект класс Person с полями – имя, возраст, пол), вес.

Реализовать:

• метод добавления животного в общий список (учесть, что добавление
  дубликатов должно приводить к исключительной ситуации)

• поиск животного по его кличке (поиск должен быть эффективным)

• изменение данных животного по его идентификатору

• вывод на экран списка животных в отсортированном порядке.
  Поля для сортировки – хозяин(имя, по возрастанию),
  если имена хозяев одинаковые - кличка животного.
  Если и имя хозяина и кличка животного одинаковые - раньше вывести
  животное, у которого больше веc.
